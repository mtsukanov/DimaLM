Как подключить за минуту (бесплатно):
  1. Откройте aistudio.google.com/apikey и войдите через Google.
  2. Нажмите Create API key, выберите проект, скопируйте ключ.
  3. Вставьте ключ в поле API-ключ ниже и нажмите Сохранить.
Ключ хранится только в вашем браузере (localStorage), никуда не отправляется. Используется бесплатная модель Gemini 2.5 Flash Lite (1000 запросов в сутки).
Расширенные настройки (URL, модель, температура)
Альтернативы AI Studio: Groq — URL https://api.groq.com/openai, модель llama-3.1-8b-instant, ключ на console.groq.com/keys (учтите ограничение 6–12K токенов/мин — большой промпт может не влезть). Локальный LM Studio — URL http://127.0.0.1:1234, поле «API-ключ» оставьте пустым.
Enter — отправить · Shift+Enter — перенос
