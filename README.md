# DimaLM

Выложить в папку, запустить:

python -m http.server 8000  

Открыть http://localhost:8000/ в браузере и получать удовольствие. 


Тестировал на LM Studio модели google/gemma-4-26b-a4b.


Добавить новые реплики:

Меняете таблицу триггеров в system_prompt.md → python3 build.py → перезагружаете страницу.
Без LM Studio: новый офлайн-скоринг подбирает фразу из таблицы по триггерам.
С LM Studio + JSON schema: приложение берёт ответ модели один-в-один, аватар точно по line_id.
